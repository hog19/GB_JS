// Самостоятельно разобраться с атрибутами тега script (async и defer). // атрибут async дает возможность страницы прогружаться без ожидания скрипта, то есть, если скрипт медленный или мешает // полноценной работе странице (например, не грузит ее), то async позволяет избежать "неправильной" работе. // атрибут defer указывает, что скрипт выполнится только тогда, когда страница загрузится полностью // (только для внешних скриптов, которые загружаются с использованием атрибута src). // не рекомендуется использование двух атрибутов, сейчас брауезеры используют только атрибут async
